[FR]
Java - Personnages 
Décembre 2012 
CF uml_perso.jpg

Jeu dans lequel des personnages (aussi appelés "Victimes") s’affrontent. 
Monstres, sorciers, gnomes, magiciens. 

Les regles du jeu sont les suivantes :
Un personnage a un nom et des points de vie. 
Si le nombre de points de vie est negatif ou nul, alors il est mort. 

Les personnages frappent (avec des coups ou des sorts)
et reçoivent (des coups ou des sorts) 

Le programme principal crée des personnages et les font s'attaquer. 
On affiche les résultats des frappes et le perdant

NOTE : une autre version avec un menu utilisateur pourrait être élaborée

[ENG] 
Java - Characters 
December 2012
SEE : uml_perso.jpg 

A game in which characters (also called Victims) fight against each other :
Monsters, wizards, gnoms, magicians 

The rules are composed as follows : 
A character has a name and hit points.
It dies when its number of hit points is equal to 0 or negative

The characters beat or cast spells 
and they receive beats or spells 

The main program creates characters and makes them fight. 
It displays the result after each beats and the name of the loser

NOTE : another version with a user menu could be built
